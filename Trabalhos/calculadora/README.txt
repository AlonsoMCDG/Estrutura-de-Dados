Descrição do trabalho:

    Faça uma calculadora que realize operações com números
de tamanho qualquer (podem ser muito grandes, p.ex.: 10^1000).
    As operações básicas que devem ser implementadas são:
        - Adição
        - Subtração
        - Multiplicação
        - Divisão
    Sinta-se livre para implementar outras operações, como
potenciação, radiciação, resto da divisão (módulo) etc.
